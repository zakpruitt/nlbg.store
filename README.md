# nlbg.store
A web application that facilitates Next Level Billiards Gear buying/selling service. This website will be live on https://nlbg.store.

# Technology Used
- Back-End
    - Java 1.8
    - Spring Boot 2.3.4
- Front-End
    - HTML5
    - CSS3
    - JavaScript
    - Bootstrap 5
- Database
    - MySQL
    - SQLite
    - PostgreSQL
- Deployment
    - Heroku
